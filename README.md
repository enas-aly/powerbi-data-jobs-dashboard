# Data Jobs Dashboard - Power BI

##  Project Overview
This project analyzes the global data job market using Power BI.  
It explores job demand, salaries, platforms, and benefits for data roles.

##  Business Questions
- How many data jobs are available?
- Which job titles are most in demand?
- What are the median salaries?
- Which platforms post the most jobs?
- How common are remote work and health insurance benefits?

##  Key Insights
- Total job postings: 479K+
- Median yearly salary: $113K
- Median hourly salary: $47.6
- Data Engineer and Data Scientist roles have higher salary ranges
- LinkedIn is the top platform for job postings
- Remote work and health insurance benefits are frequently mentioned
